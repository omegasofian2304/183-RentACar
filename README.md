# 183-RentACar

## Project Description

This project is part of the **I183 module**, where we are developing a web application for online car rental.

The primary goal of this module is to implement and understand application security. For this project, we are focusing on the Top 10 OWASP security vulnerabilities, ensuring that our application is protected against common threats such as injection attacks, broken authentication, sensitive data exposure, and more.

Through this project, we aim to combine practical web development with strong security practices, creating a functional and secure online car rental platform.

## Stack

- **Backend:** FastAPI  
- **Frontend:** Vue.js  
- **Styling:** Tailwind CSS  
- **Database:** MySQL
- **Security Focus:** OWASP Top 10 Vulnerabilities
